# Exame_EDII_G7_T3
Visão Geral: Sistema Geoespacial de Mapeamento e Alocação de Recintos Desportivos desenvolvido na linguagem C, utilizando ficheiros de texto (.txt) para persistência de dados. O sistema tem como objetivo permitir o cadastro de recintos desportivos, localização geográfica, pesquisa por proximidade, gestão de reservas e consulta de disponibilidade.

Objetivo:
    • Gerir uma lista ligada de recintos desportivos com pesquisa eficiente por nome (Trie)
    • Registar utilizadores georreferenciados com as suas coordenadas GPS
    • Gerir reservas com deteção automatica de conflitos horarios
    • Modelar a rede de caminhos como um grafo ponderado (distancias Haversine)
    • Calcular o caminho mais curto entre dois pontos com o algoritmo de Dijkstra
    • Persistir todos os dados em ficheiros de texto csv
